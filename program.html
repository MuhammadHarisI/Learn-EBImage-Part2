EBImage

EBImage menyediakan berbagai fungsi dan tujuan umum untuk pemrosesan dan analisis gambar. Pada kesempatan kali ini, kita akan belajar tentang bagaimana pemrosesan gambar, manipulasi spektral, dan pemfilteran menggunakan EBImage.

5. Manipulasi Gambar

Dengan menggunakan array numerik, gambar akan dengan mudah dimanipulasi oleh salah satu operator aritmatika dalam R. Contohnya, kita dapat menghasilkan gambar negatif hanya dengan mengurangi nilai gambar dari maksimumnya. Sebagai contohnya adalah :

img_neg = max(img) - img
display( img_neg )

Selanjutnya, Kita juga bisa meningkatkan kecerahan pada gambar dengan melalui penambahan,menyesuaikan kontras dengan perkalian dan menerapkan koreksi gamma melalui eksponen

img_comb = combine(
  img,
  img + 0.3,
  img * 2,
  img ^ 0.5
)

display(img_comb, all=TRUE)

Selanjutnya, Pada percobaan ini kita mencoba menggabungkan beberapa gambar kedalam satu objek gambar dengan multi bingkai.Kemudian kita juga dapat memotong dan membuat gambar thresholding dengan operasi matriks standart.

colorMode(Image) <- Color
display(Image[50:300, 2:300,])

Selanjutnya, Untuk melakukan transposisi gambar, anda dapat menggunakan fungsi transpose dalam pake R’s R (base") dengan fungsi t.hal ini juga dapat berfungsi pada gambar berwarna dan multiframe dengan cara menukar dimensi spasialnya, contohnya adalah :

img_t = transpose(img)
display( img_t )

6. Transformasi Spasial

kita sudah mengetahui bagaimana jenis-jenis transformasi spasial dan transposisi, namun jenis-jenisnya masih banyak lagi misalnya translasi,rotasi,refleksi,dan penskalaan. Translasi memindahkan bidang gambar dengan vektor dua dimensi yang ditentukan sedemikian rupa sehingga piksel yang berakhir di luar wilayah gambar akan dipotong,dan piksel yang masuk ke wilayah gambar diset ke latar belakang. # A. Translasi Gambar{.tabset.tabset-fade}

img_translate = translate(img, c(100,-50))

B. Rotasi Gambar

Pada merotasi gambar, Warna latar belakang dapat disetel menggunakan bg.co; yang umum untuk digunakan pada semua fungsi transformasi spasial yang relevan.lalu untuk default, disini mengatur nilai piksel latar belakang menjadi ke nol dengan warna hitam. Berikut penggunaan rotate :

img_rotate = rotate(img, 30, bg.col = "white")
display(img_rotate)

C. Mengubah Ukuran Gambar

Untuk mengubah ukuran gambar ke dimensi yang kita inginkan, kita dapat mengunakan pengubahan ukuran menggunakan cara berikut:

Contohnya :

display(img_resize )

D. Merefleksikan gambar

Menggunakn Fungsi flip dan flop untuk merefleksikan gambar di sekitar sumbu horizontal dan vertikal masing-masing menggunakan cara berikut:

7. Memberikan Filter

Pada r Biocpkg (“EBImage”), konvolusi 2 dimensi dapat diimplementasikan oleh fungsi filter-filter dan fungsi tambahan makeBrush itu dapat digunakan untuk menghasilkan fungsi bobot. Contohnya :

w = makeBrush(size = 31, shape = 'gaussian', sigma = 5)
plot(w[(nrow(w)+1)/2, ], ylab = "w", xlab = "", cex = 0.7)

Kemudian, dibawah ini untuk mendeteksi tepi dan mempertajam gambar

fhi = matrix(1, nrow = 3, ncol = 3)
fhi[2, 2] = -8
img_fhi = filter2(img, fhi)

Selanjutnya, Operasi morfologi Gambar biner adalah gambar yang hanya berisi dua set piksel, dengan nilai, katakanlah 0 dan 1, mewakili piksel latar belakang dan latar depan. Contohnya adalah :

10. Segmentasi Sel

Pada hal ini, Kita bisa memuat gambar inti dan badan sel. Lalu Untuk memvisualisasikan sel, kita dapat melapisi gambar-gambar ini sebagai saluran hijau dan biru dari gambar berwarna palsu dengan cara berikut:

nuc = readImage(system.file('images', 'nuclei.tif', package='EBImage'))

cells = rgbImage(green=1.5*cel, blue=nuc)

Contohnya :

display(cells, all = TRUE)

Yang Pertama, kita segmentasi menggunakan thresh, fillHull, bwlabel dan opening dengan cara sebagai berikut:

nmask = thresh(nuc, w=10, h=10, offset=0.05)
nmask = opening(nmask, makeBrush(5, shape='disc'))

nmask = bwlabel(nmask)

Contohnya :

display(nmask, all=TRUE)

Kedua, kita akan menggunakan inti tersegmentasi sebagai benih dalam segmentasi Voronoi dari sitoplasma dengan cara berikuy:

ctmask = opening(cel>0.1, makeBrush(5, shape='disc'))

Contohnya :

Lalu, Untuk memvisualisasikan segmentasi kita pada yang kita gunakan paintObject

11. Pemrosesan Gambar dan Representasi Spektral Sinyal

# panggil package
library(EBImage)

# read image
image <- readImage('E:/file desktop/Perkuliahan/Perkuliahan Semester 2/Linear Algebra/Data Science and Predictive Analytics/Masjid.JPG')
display(image, method = "raster", all = TRUE)

# Intensity Histogram
hist(image)

# Generate negative image
img_neg <- max(image) - image
display( img_neg )

# dim: 633 1000    3

# crop and threshold
img_crop <- image  
img_thresh <- img_crop > 0.55
display(img_thresh)

# Low-pass Filter/smooth
w <- makeBrush(size = 31, shape = 'gaussian', sigma = 5)
img_flo <- filter2(image, w)
display(img_flo)

# High-pass filtering Laplace Filter/Edge detection
fhi <- matrix(1, nrow = 3, ncol = 3)
fhi[2, 2] <- -7
img_fhi <- filter2(image, fhi)
display(img_fhi)

# Denoising
img_median <- medianFilter(image, size=5, cacheSize=20000)
display(img_median)

# Segmentation
grayimage <- channel(image,"gray")
threshold <- otsu(grayimage)
CC_th <- combine( mapply(function(frame, th) frame > th, 
                         getFrames(image), 
                         threshold, SIMPLIFY=FALSE) )
display(CC_th, all=TRUE)

disc <- makeBrush(51, "disc")
disc <- disc/sum(disc)
offset <- 0.0001
CC_bg <- filter2(image, disc)
CC_th <- image < CC_bg + offset
display(CC_th, all=TRUE)

# Watershed segmentation
nmask <- watershed(distmap(CC_th), tolerance=5, ext=20)
display(colorLabels(nmask), all=TRUE)

# Voronoi image tesselation
voronoiExamp <- propagate(x = nmask, seeds = nmask, lambda = 0.001)
voronoiPaint <- colorLabels (voronoiExamp)
display(voronoiPaint, all=TRUE)

CC <- rgbImage(green=2.0*CC_bg, blue=img_median, red=voronoiPaint[ , , , 1])
display(CC, all = TRUE)

ctmask = opening(img_fhi<0.6, makeBrush(5, shape='disc'))
cmask = propagate(img_fhi, seeds=nmask, mask=ctmask)
display(ctmask, all=TRUE)

ctmask = opening(img_fhi>0.1, makeBrush(5, shape='disc'))
cmask = propagate(img_fhi, seeds=nmask, mask=ctmask)
display(ctmask, all=TRUE)

segmented = paintObjects(channel(cmask,"gray"), image, col='#ff00ff')
segmented = paintObjects(channel(ctmask,"gray"), segmented, col='#ffff00')
display(segmented, all=TRUE)
